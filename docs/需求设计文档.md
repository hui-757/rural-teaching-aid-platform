# 乡村教学辅助平台 — 需求设计文档

> **目标用户**：农村小学教师（当前仅支持四年级）
> **核心痛点**：自主出题困难、课堂互动枯燥、学生上课缺乏兴趣
> **核心解决方案**：教材辅助 + 基础练习 + 课堂互动 + 闯关游戏

---

## 一、项目背景

乡村教师在教学过程中面临三大困难：
1. **出题困难**：难以自主设计符合教学内容的题目
2. **课堂互动**：需要围绕学生回答持续出题，缺乏即时可用的题库
3. **学生兴趣**：传统教学方式难以吸引学生注意力，课堂参与度低

本平台通过提供**教材辅助**、**基础练习**、**课堂互动**、**闯关游戏**四大功能，帮助教师降低备课负担，提升课堂互动性，并通过游戏化机制激发学生学习兴趣。

---

## 二、用户角色

| 角色 | 说明 |
|------|------|
| **教师（用户）** | 平台的主要使用者，负责授课、出题、组织闯关活动 |
| **学生** | 闯关游戏的参与者，在教师引导下使用平台闯关 |

> 注：学生无需注册账号，通过教师发起的闯关活动参与即可。

## 四、授课模块

教师进入"授课"后，需先选择单元。每个单元提供独立页面。

### 4.1 单元选择页
- 列出所有可选单元（教材结构）
- 每个单元卡片包含：单元名称、单元简介、是否支持基础测试标识

### 4.2 单元详情页（三栏布局）

```
┌──────────────────┬──────────────────┬──────────────────┐
│   内容讲述        │    基础测试       │     一起练        │
│  (教材内容)       │  (题库题目)       │  (课堂互动)       │
│                  │                  │                  │
│  · 直接呈现教材   │  · 输出计算题    │  · 学生举手回答   │
│    对应内容       │  · 可打印文档    │  · 巩固练习       │
│  · 辅助教师讲解   │  · 答案单独存放  │  · 趣味较低       │
│                  │                  │   侧重交互        │
└──────────────────┴──────────────────┴──────────────────┘
```

#### 4.2.1 内容讲述
- 直接呈现教材对应单元的教学内容
- 辅助教师讲解，无需额外准备教材

#### 4.2.2 基础测试
- **功能**：从题库中抽取对应单元的计算题，供课堂练习或课后作业
- **输出格式**：Word / PDF（便于打印）
- **答案处理**：答案**不随题目一同打印**，单独存放于**教师用户中心**
- **答案记录**：标注来源信息——
  - 生成时间
  - 所属单元
  - 第几次测试（序号）

> **限制**：仅面向含有计算题目的单元提供基础测试功能。不含计算题的单元不显示该入口。

#### 4.2.3 一起练
- **功能**：课堂互动环节，学生举手回答，巩固所学知识
- **特点**：趣味较低，侧重师生实时互动

## 五、竞赛（闯关）模块

教师进入"竞赛"后，组织学生在课堂中进行闯关游戏。

### 5.1 流程概览

```
选择单元 → 开始闯关 → 学生上台参与 → 保存记录 → 备注 → 排行展示
```

### 5.2 功能设计

| 功能 | 说明 |
|------|------|
| **单元选择** | 闯关前需选择对应单元（与授课单元一致） |
| **关卡设置** | 每个单元设置约 **10 个关卡** |
| **首次定级** | 学生首次进入第 1 关时，系统根据表现自动定级 |
| **闯关记录** | 每次闯关自动保存记录 |
| **闯关备注** | 每次闯关结束后，弹窗提示教师进行备注（如学生表现、注意事项） |
| **进度排行** | 展示各单元闯关进度排行榜 |
| **重置闯关** | 教师可随时重置闯关进度，方便其他学生重新开始（适应乡村课堂多班级/多轮次场景） |

> **场景说明**：闯关由教师发起，学生轮流上台参与。教师可在用户中心查看历史闯关记录及备注。

## 六、页面结构设计

### 6.1 首页
```
顶部导航栏
├── [年级选择]
├── [用户昵称]
└── [用户中心入口]

主体内容
├── 授课入口（进入单元选择）
└── 竞赛入口（进入闯关）
```

### 6.2 授课 — 单元选择页
```
单元列表
├── 单元 1 卡片 [内容讲述] [基础测试（如有）] [一起练]
├── 单元 2 卡片 ...
└── ...
```

### 6.3 授课 — 单元详情页
```
三栏布局
├── 左栏：内容讲述（教材内容展示）
├── 中栏：基础测试（题库题目输出 + 打印按钮）
└── 右栏：一起练（课堂互动环节）
```

### 6.4 竞赛 — 闯关页
```
单元选择 → 关卡列表
├── 关卡 1 ~ 关卡 10（每个单元约10关）
├── 闯关进行（学生上台）
├── 结束弹窗：备注输入
├── 保存记录
└── 排行展示
```

### 6.5 用户中心
```
教师个人中心
├── 基础测试答案列表（按时间/单元/序号分类）
└── 闯关历史记录（含备注）
```

---

## 八、总结

本平台的核心价值在于**降低农村教师的备课负担**，通过以下路径实现：

1. **教材辅助** → 无需额外准备教材内容，直接讲解
2. **基础测试** → 一键生成可打印的练习题，答案单独管理
3. **一起练** → 课堂即时互动，巩固知识
4. **闯关游戏** → 游戏化激发学生兴趣，同时可记录、可重置、可排行，适应乡村课堂多轮次使用场景
